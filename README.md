# 1. 프로젝트
여가생활 기반 노인치매 예방을 위한 정책 방향 제시

# 2. 주제 선정 배경

노인인구 증가에 따라 노인 우울, 치매 등 노인 건강 문제가 중요해지고 있음, 노인 치매 예방을 위한 효과적인 정책에 대한 필요성 증가

# 3. 목적
